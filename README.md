# MockFoundry

Generador de datos fake para testing y seeds, con campos personalizados y preajustes típicos de aplicaciones web.

- Features
  - Selección de campos comunes: nombre, email, teléfono, ciudad, texto lorem, etc.
  - Generación de múltiples registros y descarga como JSON
  - Presets para distintos contextos (usuarios, productos, posts, etc.)

- Stack
  - React
  - TypeScript
  - Tailwind CSS
